# SDA Taman Harapan Church Website

**Tagline**: "Menanti Kedatangan-Nya, Melayani Sesama"

## About
Official website for SDA Taman Harapan Church - A modern, bilingual (Indonesian & English) church website.

## Features
- Bilingual support (ID/EN)
- Sermon archive
- Events calendar
- Ministries information
- Sabbath School resources
- Online giving
- Contact forms

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
